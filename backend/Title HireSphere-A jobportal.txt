Title: HireSphere-A jobportal

technologies used:
 **React 19**
- **React Router DOM 7**
- **Axios** 
- **Node.js**
- **Express**
- **MongoDB** with **Mongoose**
- **JWT** for authentication
- **Multer** for image uploads
- **Dotenv** for environment configuration

description: A interface between a jobseeker and and recruiter.where a jobseeker will able to apply the jobs which are posted by the recruiter.and recruiter will analyse the application and give response

features:
 JWT-based authentication
 Separate models for Job Seekers and Employers
 Job Posting and Management
 Job Application System with status tracking
 Stats APIs for dashboard displays
 File upload support via Multer
